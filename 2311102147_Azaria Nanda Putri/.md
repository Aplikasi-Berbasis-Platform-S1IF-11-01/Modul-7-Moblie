# 1. Pendahuluan

Flutter merupakan framework UI open-source dari Google yang digunakan untuk membangun aplikasi multiplatform menggunakan satu basis kode. Flutter menggunakan bahasa pemrograman Dart dan menerapkan konsep widget untuk membentuk tampilan aplikasi.

Pada praktikum ini dibuat aplikasi Flutter bertema pink calm dengan nama **Data Mahasiswa**. Aplikasi ini memiliki beberapa halaman utama seperti:

1. Home Page
2. Form Input Data Mahasiswa
3. Profile Developer

Aplikasi menerapkan konsep:

* Google Fonts
* Form Validation
* StatefulWidget dan StatelessWidget
* Navigator
* SnackBar
* Material Design 3

Tema aplikasi menggunakan kombinasi warna pink feminin dengan font Poppins agar tampilan lebih modern, elegan, dan nyaman dilihat.

# 2. Dasar Teori

## 2.1 Flutter

Flutter adalah framework UI berbasis widget yang dikembangkan oleh Google untuk membangun aplikasi Android, iOS, Web, dan Desktop.

## 2.2 Dart

Dart merupakan bahasa pemrograman yang digunakan Flutter. Program Flutter dimulai dari fungsi `main()` dan dijalankan menggunakan `runApp()`.

```dart
void main() {
  runApp(const MyApp());
}
```

## 2.3 Google Fonts

Google Fonts digunakan untuk mempercantik tampilan teks pada aplikasi.

```dart
textTheme: GoogleFonts.poppinsTextTheme(),
```

## 2.4 StatefulWidget

`StatefulWidget` digunakan ketika tampilan membutuhkan perubahan data secara dinamis.

```dart
class FormMahasiswaPage extends StatefulWidget {
  const FormMahasiswaPage({super.key});

  @override
  State<FormMahasiswaPage> createState() => _FormMahasiswaPageState();
}
```

## 2.5 StatelessWidget

`StatelessWidget` digunakan untuk tampilan yang tidak berubah.

```dart
class HomePage extends StatelessWidget {
  const HomePage({super.key});
}
```

## 2.6 TextEditingController

`TextEditingController` digunakan untuk mengambil data dari input user.

```dart
final TextEditingController _namaController = TextEditingController();
```

## 2.7 Form Validation

Validasi form dilakukan menggunakan `GlobalKey<FormState>()`.

```dart
final _formKey = GlobalKey<FormState>();
```

## 2.8 Navigator

Navigator digunakan untuk berpindah halaman.

```dart
Navigator.push(
  context,
  MaterialPageRoute(builder: (_) => HalamanBaru()),
);
```

## 2.9 SnackBar

SnackBar digunakan untuk memberikan notifikasi kepada pengguna.

```dart
ScaffoldMessenger.of(context).showSnackBar(
  SnackBar(content: Text('Data berhasil disimpan')),
);
```

## 2.10 Material Design 3

Material Design 3 digunakan untuk menciptakan tampilan modern dengan color scheme yang konsisten.

# 4. Langkah-Langkah Praktikum

## 4.1 Membuat Project Flutter

```bash
flutter create flutter_mahasiswa
cd flutter_mahasiswa
```

## 4.2 Menambahkan Dependency

Tambahkan package Google Fonts pada file `pubspec.yaml`.

```yaml
dependencies:
  flutter:
    sdk: flutter
  google_fonts: ^6.2.1
```

Lalu jalankan:

```bash
flutter pub get
```

## 4.3 Struktur Folder

```text
lib/
├── main.dart
├── home_page.dart
├── form_mahasiswa_page.dart
└── profile_page.dart
```

## 4.4 Membuat main.dart

File `main.dart` digunakan untuk:

* Menjalankan aplikasi Flutter
* Mengatur tema global aplikasi
* Menggunakan Google Fonts Poppins
* Mengatur warna pink calm pada aplikasi

Warna utama yang digunakan:

```dart
seedColor: const Color(0xFFE8739A)
```

## 4.5 Membuat home_page.dart

Halaman utama menampilkan:

* Hero banner gradient pink
* Menu navigasi
* Tombol menuju form mahasiswa
* Tombol menuju profile developer

Home page dibuat menggunakan `StatelessWidget`.

## 4.6 Membuat form_mahasiswa_page.dart

Halaman form digunakan untuk:

* Input nama mahasiswa
* Input NIM
* Input kelas
* Validasi form
* Menampilkan hasil data
* Menampilkan SnackBar ketika data berhasil disimpan

Halaman ini menggunakan `StatefulWidget` karena data dapat berubah.

## 4.7 Membuat profile_page.dart

Halaman profil developer berisi:

* Nama developer
* NIM
* Program studi
* Deskripsi diri
* Skills
* Hobi

Tampilan menggunakan desain pink feminine dengan icon bunga.

## 4.8 Menjalankan Program

```bash
flutter run
```
